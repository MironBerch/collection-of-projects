# Social_network [![made-with-python](https://img.shields.io/badge/Made%20with-Python-1f425f.svg)](https://www.python.org/)

## Technologies:
- python 3.10
- Django 4
- Django REST framework
- PostgreSQL
- Nginx
- Docker
- docker-compose

## Start project:

### by using docker on local server
```
docker-compose up --build
```
